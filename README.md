PPGo_Logstah
====
什么东西？What?
----
一款用于nginx日志监控的开源方案。
读取NGINX日志->存入InfluxDb->Grafana查看数据

监控指标：nginx流量，响应时间，访问地址等

基于golang


有什么价值？
----
1、监控nginx日志
2、influxDb连接池实现

用到了哪些？
----
1、golang1.8+
2、influxdb1.2
3、grafana4.2
